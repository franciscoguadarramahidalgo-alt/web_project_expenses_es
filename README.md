Nombre del proyecto: Gestor de Gastos

Descripción del proyecto: Se desarrolló un Gestor de Gastos con el objetivo de ayudar al usuario a controlar sus finanzas y mantenerse informado sobre los gastos realizados durante un periodo determinado. La aplicación ofrece una forma visual e intuitiva de identificar el estado del saldo mediante un sistema de colores: rojo (saldo negativo), amarillo (por debajo del 25 % del presupuesto inicial) y verde (saldo estable).

Además, permite al usuario visualizar el saldo total en tiempo real, identificar la categoría con mayor gasto y conocer el monto correspondiente. La página también posibilita dividir los gastos por categoría, facilitando una comprensión clara de la distribución del presupuesto asignado a cada una.

El principal objetivo de este proyecto fue implementar y poner en práctica las tecnologías de JavaScript, asegurando el correcto funcionamiento de sus componentes dentro del Gestor de Gastos.
